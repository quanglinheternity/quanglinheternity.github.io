# 💻 Personal Portfolio Website - Nguyen Quang Linh

This is a **responsive multi-page portfolio website** showcasing my work as a Backend Developer. It features modern design elements, interactive animations, and full multi-language support (English & Vietnamese).

---

## 📂 Project Structure

```
├── index.html          → Home Page (with Typing Effect)
├── about.html          → About Me Page
├── skills.html         → Technical Skills Page
├── projects.html       → Projects Showcase (with Grid Layout)
├── style.css           → Custom Premium Styles
├── script.js           → Core Logic (Language Switching & Typing Effect)
├── translations.js     → Translation Dictionary (EN/VI)
├── main.jpg            → Display Picture
└── README.md           → Project Documentation
```

---

## ✨ Features

✔️ **Multi-language Support**: Seamless switching between English and Vietnamese.  
✔️ **Typing Effect**: Dynamic typewriter animation for the hero title.  
✔️ **Refined Justification**: Clean "căng đều" project descriptions with `inter-word` spacing.  
✔️ **Responsive Design**: Fully optimized for Mobile, Tablet, and Desktop.  
✔️ **Modern UI**: Glassmorphism effects, smooth gradients, and AOS animations.  
✔️ **Interactive Navigation**: Stylish fixed navbar with a mobile-friendly drawer menu.  

---

## 📱 Technologies Used

- **HTML5 & CSS3**: Custom styling with Flexbox and Grid.
- **JavaScript (Vanilla)**: For language management and UI interactions.
- **Font Awesome**: High-quality vector icons.
- **AOS (Animate On Scroll)**: For smooth reveal animations.
- **Google Fonts**: Modern typography ('Poppins').

---

## 🛠 How to Run Locally

1. Clone this repository:

```bash
git clone https://github.com/quanglinheternity/my_profile.git
```

2. Open the project folder:

```bash
cd my_profile
```

3. Open `index.html` in your browser.
